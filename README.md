# Zeyro Finance Tracker

A modern, feature-rich finance tracker app built with Flutter. Track your income, expenses, and view analytics with a clean, organized UI.

## Features

- Add, edit, and delete transactions
- Monthly summary and total balance
- Search and filter transactions
- 3-tab navigation (Home, Add, Analytics)
- Local storage using Hive
- Responsive design for web and mobile

## Tech Stack

- Flutter
- Provider (state management)
- Hive (local storage)
- Dart

## Setup and Run Instructions

1. **Clone the repository:**
git clone https://github.com/yourusername/zeyro_finance_tracker.git
cd zeyro_finance_tracker

2. **Install dependencies:**
flutter pub get

3. **Run the app:**
flutter run
- Select your target device (web, Windows, Android, iOS).

4. **Build for production:**
flutter build web

## Local Storage

- This app uses **Hive** for local storage.
- All transactions are saved locally and persist across app restarts.
